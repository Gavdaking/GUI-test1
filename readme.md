# Cine-Search Pro

A movie search app built using **Vanilla JavaScript, HTML, and raw CSS**.  
This project focuses on async data handling and clean frontend structure without using any frameworks.

---

## Week 1 Progress

- Created `SearchComponent` class
- Implemented debounce (300ms) using `setTimeout`
- Connected to TMDB API using `fetch()`
- Basic UI: search input and results list
- Loading state handled with `data-loading` attribute

---

## How It Works

User types → debounce → fetch API → display results

---

## Tech Used

- HTML
- CSS
- Vanilla JavaScript
- TMDB API
